# jogo-da-confian-a
Um jogo feito em python pra testar a confiança dos seus amigos
o jogo consiste em escolher quantos jogadores irão jogar, e um jogador será sorteado com uma "arma" carreagada e poderá escolher entre descaregar a arma e deixa-la carregada
após essa escolha a pessoa entregará a arma para outro jogador escolhido que pode atirar na propria cabeça (confiando que a arma está descarregada) ou atirar na pessoa que lhe entrgou a arma
